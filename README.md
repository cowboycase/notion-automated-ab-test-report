# Notion Automated A/B Test Report 📊

This project automatically analyzes the results of an A/B test from a CSV file and generates a comprehensive report on a designated Notion page. The entire process, from data analysis to report generation, is handled by a single Python script.

## 🎯 Project Goal

The main goal of this project was to showcase skills in data analysis, statistical testing, and API integration, specifically tailored for a company like Notion. It solves a common business problem (analyzing A/B test results) and uses Notion as the final reporting dashboard, demonstrating both technical ability and a genuine interest in the product.

---

## 🛠️ Technologies Used

*   **Python**
*   **Pandas:** For data manipulation and analysis.
*   **SciPy:** For performing the Chi-Squared statistical test.
*   **Matplotlib & Seaborn:** For data visualization.
*   **Notion API:** To programmatically create and update the report page.
*   **notion-client:** The official Python library for the Notion API.

## 🚀 How to Run

1.  Clone this repository.
2.  Install the required libraries:
    ```bash
    pip install -r requirements.txt
    ```
3.  Open the `ab-test-analyze.py` script and fill in your own `NOTION_API_KEY`, `NOTION_PAGE_ID`, and a direct image link for the graph.
4.  Run the script:
    ```bash
    python ab-test-analyze.py
    ```
5.  Check your Notion page! The report will be generated automatically.
